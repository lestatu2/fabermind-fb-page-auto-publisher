Automatically publish WordPress posts and selected custom post types to a Facebook Page via the Meta Graph API.

== Description ==

This plugin publishes newly published content to a Facebook Page using a manually configured Page ID and Page Access Token.

Features:
- Facebook Pages only
- Select supported post types in settings
- Per-post metabox override
- 1200x630 image generation from featured image
- `/photos` first, `/feed` fallback
- Basic logging through post meta

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`.
2. Activate the plugin from WordPress admin.
3. Go to `Settings > Facebook Auto Publisher`.
4. Configure Page ID, Page Access Token, and supported post types.
